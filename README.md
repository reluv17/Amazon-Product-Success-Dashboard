# Amazon Product Success Prediction Dashboard

Predicting product success from early review patterns using machine learning.

## Features
- Early review predictive analysis (r=0.976 correlation)
- Volatility-based warning system
- Category-specific risk profiles
- Interactive dashboard with 1.5M reviews analyzed

## Data
- 1.5M Amazon reviews (2015-2023)
- 471 products analyzed
- Categories: Beauty, Electronics, Pet Supplies

## Models
- Random Forest: 96.8% accuracy
- Logistic Regression: 95.8% accuracy  
- Gradient Boosting: 95.8% accuracy
